2d image of 3d objects

Explaination of files:
1. main.py -- main python source file
2. objs/ -- folder containing input objects
3. outputs/ -- folder containing output svg's
4. runner.sh -- bash script containing inputs for the output svg's

Running:
Simply run as:
python main.py -h
to know all arguments and parameters.

Examples:
Using obj inputs in objs/ folder, and commands shown in runner.sh script, some example outputs are in the outputs/ folder. Go through runner.sh script to get an idea of some possible runs.
